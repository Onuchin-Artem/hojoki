# Web edition — progress & how to resume

Web version of «Думки у ретрітній хатинці» (Ukrainian Hōjōki). Separate screen
artifact, not a print export. Spec: `../hojoki-html.md`.

## Build & preview
```
python3 web/build_web.py --stage 1     # prototype subset; --stage 2 = whole book
cd web && python3 -m http.server 8000  # → http://localhost:8000
```
Outputs (all in `web/`): `index.html` + `style.css` + `assets/img/` (responsive
WebP+JPEG, generated by the build). Source of truth = `../source/*.md`
(`hojoki.md`, `footnotes.md`, `captions.md`); Typst remains the markup-semantics
reference. No pandoc.

Dev screenshots: `web/_shots_pw.py` (Playwright, chromium). Plain
`chrome --headless` is unreliable below 500px — use Playwright with
`device_scale_factor=2, reduced_motion="reduce"`.

### Layout decisions (these OVERRIDE the spec where they differ)
- **Frame:** warm light block (`--bg #fdfcf8`), **fixed 720px, centred**, on a
  **dark-paper field** (`--page #15110d` + faint SVG grain). Full-width < 900px.
- **No breathing pauses** (spec §2 dropped). Chapters flow; `.chapter` top padding
  = 2× bottom.
- **All titles left-aligned.** Prose **left-aligned, no justify, no hyphenation.**
- **Content padding** via `--pad-l/--pad-r`: desktop 50/75px; mobile (≤640px)
  33/33px (equal) and root font scaled to **85%**, with title −/name overrides.
- **Title page** (spec said no author/translator — overridden): red NBSP-bound
  «Камо но Тьомей» on top, big one-line title, grey right-aligned
  «Переклад з англійської — Артем Онучін» below. Author/translator NBSP-bound via
  `bind_names()`.
- **Contents** = `<details>` «Зміст», collapsed, inline in the support row; lists
  chapters + Примітки only (no front matter).
- **Marginalia** (desktop footnote) = warm panel colour; shows prose only — the
  italic **lemma quote lives only in the appendix** and keeps its verse line breaks.
- **Markers:** ignore `---`/`===` pagebreaks and `~Nmm` last-page skips; a **bare
  `~`** is kept as an empty-paragraph gap; `>2em` stanza indent kept.
- Donate → `https://savelife.in.ua/donate`; contact «Написати перекладачу» →
  `onuchinart@gmail.com`. Click tracking = Plausible custom event `Donate`
  (`tagged-events`); `PLAUSIBLE_DOMAIN` is still a `{{...}}` placeholder.
